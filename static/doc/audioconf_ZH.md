### 参数

没有服务调用参数会导致语音转文字不可用
[语音模型](https://www.volcengine.com/docs/6561/80816)

| Parameter Name  | Type     | Required/Optional | Description  |
|-----------------|----------|-------------------|--------------|
| `AUDIO_APP_ID`  | `string` | 可选                | appid        |
| `AUDIO_TOKEN`   | `string` | 可选                | access token |
| `AUDIO_CLUSTER` | `string` | 可选                | cluster id   |
